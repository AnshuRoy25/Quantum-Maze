In General 
Use W/A/S/D Keys for movements

Special Cases:
Level 2: Use 'F' Button to collapse the superposition state of the entangled doors. This F button will only work when you are adjacent to a door in superpostion.
Level 3: Use W/A/S/D + SpaceBAR to Tunnel in that particular direction through the wall. Eg: For Tunneling through Right wall, use (D + Spacebar)
